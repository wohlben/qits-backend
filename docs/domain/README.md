# QITS

QITS (Quickly Iterate The Software) is an application built to accelerate software development workflows through rapid iteration.

This repository provides high-level domain documentation for the QITS ecosystem. The application is composed of multiple technical components, currently:

- **qits-ui** — the frontend interface
- **qits-backend** — the backend services

Additional components may be split out as the system evolves.

## Documentation

Domain documentation lives in [`docs/`](docs/). It is organized by user flows and describes what the system does and why, independent of any single implementation.
